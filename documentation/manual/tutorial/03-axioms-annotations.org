- =rdfs:label= and =skos:definition= annotations on every class,
- a =disjointWith= axiom between =ex:Dog= and =ex:Cat=,
- an object property =ex:chases= with =rdfs:domain= and =rdfs:range=,
- an existential restriction: every =ex:Dog= /chases some/ =ex:Cat=.

* Annotations

Use the =:annotations:= drawer key to attach annotations to a class:

#+begin_src org :exports code
,*** ex:Dog
,:PROPERTIES:
,:subclass-of: ex:Animal
,:END:
,:annotations:
,- rdfs:label :: "Dog"@en
,- skos:definition :: "A domesticated canine."@en
,:end:
#+end_src
