* Ontology heading properties

Each ontology heading (the heading carrying =:ELOT-context-type:
ontology=) declares its identity and authoring conventions in its
=:PROPERTIES:= drawer.  The principal properties are:

| Property                  | Purpose                                                    |
|---------------------------+------------------------------------------------------------|
| =:ELOT-context-type:=     | Must be =ontology= for the heading to be recognised.       |
| =:ELOT-context-localname:= | Short name used for tangle targets and IDs.               |
| =:ELOT-default-prefix:=   | CURIE prefix used for resources declared in this ontology. |
| =:ELOT-id-scheme:=        | Identifier-minting scheme used by ELOT's authoring tools.  |
