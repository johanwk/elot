#+TITLE: elot-cli Implementation Plan
#+DESCRIPTION: Port of ELOT Org->OMN pipeline from Emacs Lisp to TypeScript CLI

* Overview

Port the core ELOT functionality — reading an Org-mode outline and generating
OWL Manchester Syntax (OMN) — from Emacs Lisp (=elot-tangle.el=) to a
standalone TypeScript CLI using =orga= for Org parsing.

Reference Elisp source: =elot-elisp/elot-tangling.org=
Golden test fixture: =examples/bfo-core.org= -> =examples/bfo-core.omn=
